## Files

- `index.html` - 網站主頁內容與社群分享 metadata
- `styles.css` - 頁面樣式
- `assets/` - 網站使用的圖片素材
- `source-profile.json` - 公開資料整理
- `source-notes.md` - 資料來源與整理筆記

## Development

這是純靜態網站，不需要 build step。直接開啟 `index.html`，或用任何靜態伺服器提供專案根目錄即可預覽。
